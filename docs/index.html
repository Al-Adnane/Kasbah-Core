The fortress opens

Before any tool executes, the agent asks permission. We calculate integrity, sign a cryptographic ticket, hand it back. The ticket works ONCE. 

// Agent requests entry

  tool: 'db.query',
  agent_id: 'claude',
  signals: { 
    consistency: 0.95,
    accuracy: 0.98
